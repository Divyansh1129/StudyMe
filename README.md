## Overview

StudyMe is a machine learning project that predicts students' final exam scores based on their weekly study hours using **Linear Regression**. It demonstrates the complete machine learning workflow, from data preprocessing and visualization to model training, evaluation, and prediction.

Designed as a foundation for future AI-powered educational tools, StudyMe aims to help students better understand and improve their academic performance through data-driven insights.

---

## Features

- Student score prediction using Linear Regression
- Data preprocessing with Pandas
- Performance evaluation using MAE, MSE, RMSE, and R² Score
- Data visualization using Matplotlib
- Histogram of score distribution
- Regression line with actual data points
- Predict scores for custom study hours

## Model Evaluation

The model is evaluated using:

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)

- R² Score
